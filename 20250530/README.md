## ローカル環境でのテスト方法
```zsh
make test DATE=< 課題の日付: デフォルトは今日 > ASSIGNMENT=< 課題番号 (全ての場合: ALL): デフォルトは ALL >

例) 課題の当日に全ての課題に関してテストを行う場合
    make test

例) 課題の当日に課題 1 だけテストを行う場合
    make test ASSIGNMENT=1
```
